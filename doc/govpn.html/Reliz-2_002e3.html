Релиз 2.3

  • Пакеты рукопожатия стали неотличимыми от случайных данных. Теперь весь GoVPN трафик это шум для человека по середине.
  • Сообщения рукопожатия стали меньше (16% сокращения трафика).
  • Злоумышленник не может создать обманных пакеты рукопожатия которые заставят сервер генерировать приватный DH ключ, предотвращая потребление энтропии и ресурсоёмкие вычисления.
